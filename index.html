MINHAS ESPECIALIDADES

Cloud

Tecnologias Cloud, especialmente AWS, aplicando ferramentas como S3, EC2, Redshift, Glue, Athena e Kinesis para criar projetos de Big Data. Utilizo essas soluções para armazenar, processar e analisar grandes volumes de dados de forma escalável, integrando e otimizando pipelines de dados para suportar decisões estratégicas baseadas em dados.

Governança de Dados

Experiência com diversos tipos de bancos de dados, tanto SQL quanto NoSQL, e em modelagem de dados eficiente para garantir integridade e performance. Uso de ferramentas como dbt para transformar dados e Snowflake para análises escaláveis em data warehouses na nuvem. Com Databricks e Apache Spark, realizar processamento distribuído, otimizando pipelines de dados e impulsionando a tomada de decisões baseada em dados de forma ágil e eficiente.

Programação

Python, usando Pandas e NumPy para análise de dados e realizando web scraping para coleta de informações online. Essa expertise permite estruturar fluxos de trabalho eficientes, otimizando a preparação de dados para decisões estratégicas.
